#Leetcode #Problems #Arrays #Matrix
##### Problem Description
```
Given an m x n integer matrix matrix, if an element is 0, set its entire row and column to 0's.
You must do it in place.
```

##### Code
```Java
class Solution {
    public void setZeroes(int[][] matrix) {
        Set<int[]> set = new HashSet<>();

        for(int i=0; i<matrix.length; i++){
            for(int j=0; j<matrix[0].length; j++){
                if(matrix[i][j] == 0){
//you shouldn't directly set the row and col to 0 here
//otherwise, for the next loop, unneccessary row/col will hit and make entire mat 0
                    set.add(new int[]{i,j});
                }
            }
        }

        for(int[] i : set){
            setMat(matrix, i[0], i[1]);
        }
    }
    void setMat(int[][] matrix, int m, int n){
        for(int i=0; i<matrix.length; i++) matrix[i][n] = 0;
        for(int i=0; i<matrix[0].length; i++) matrix[m][i] = 0;
    }
}
```

###### Trick
Don't directly set the row and column to 0 when you see an element = 0. Because if you do so, the next loops will also satisfy `matrix[i][j] == 0` and thus make the entire matrix = 0
